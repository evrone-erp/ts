# Используемые варианты
### Обычный текст
```css
font-weight: 400;
font-size: 14px;
line-height: 22px;
color: #262626;
```
### Код

  <Story name="Обычный текст">
    <Text lh="22">Пример текста</Text>
  </Story>
</Canvas>
